**Interactive Astronomy Explorer** built for the **AstralWeb Hackathon 2026**

SkyDome is a web-based astronomy explorer that allows users to discover what is visible in the night sky from any location on Earth. Users can search any city, use their current GPS location, view visible planets and constellations, track the International Space Station (ISS), and explore a simplified real-time Solar System Radar.

# ✨ Features

### 🌍 Interactive Earth Map
- Search any city worldwide
- Use current GPS location
- Click anywhere on the map to change location
- Reset location instantly

### 🌌 Sky View
- Displays visible planets
- Displays visible constellations
- Shows current ISS position
- Live ISS altitude, latitude and longitude

### 🪐 Solar System Radar
- Animated Solar System visualization
- Includes all 8 planets:
  - Mercury
  - Venus
  - Earth
  - Mars
  - Jupiter
  - Saturn
  - Uranus
  - Neptune
- Realistic relative orbital speeds
- Responsive design for desktop and mobile

### 📱 Responsive UI
- Mobile-friendly
- Tablet-friendly
- Desktop optimized
- Space-themed interface with animated background

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/prakharnag637/SkyDome.git
```

Go into the project

```bash
cd SkyDome
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
